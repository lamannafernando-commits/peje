🔑 Token de GitHub requerido

Para sincronizar datos entre dispositivos necesitás un Personal Access Token de GitHub.

Cómo obtenerlo:
1. Entrá a github.com → tu perfil
2. Settings → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Permiso: ✅ repo | Expiración: No expiration
6. Copiá el token (empieza con ghp_)
